## 🚀 Быстрая установка

```bash
# Клонирование репозитория
git clone https://github.com/yourusername/vps-finder.git
cd vps-finder

# Запуск установки
chmod +x install.sh
./install.sh
Что произойдет во время установки:
Автоматическое определение вашей ОС

Установка Node.js (если не установлен)

Установка PM2 для управления процессом

Запрос/генерация паролей (с выбором способа)

Установка зависимостей

Запуск приложения

Пароли, которые нужно настроить:
Пароль	Назначение	Требования
Веб-пароль	Доступ к странице http://сервер:3000	минимум 8 символов
Мастер-пароль	Управление паролями на сервере	минимум 10 символов
Выбор способа создания пароля:
text
Выберите способ:
  1) Ввести пароль вручную
  2) Сгенерировать случайный пароль (16 символов)
  3) Сгенерировать читаемый пароль (легко запомнить)
🔐 Управление паролями на сервере
Просмотр паролей:
bash
# Через меню управления
./manage-passwords.sh

# Или напрямую
cat data/web-password.txt
Смена пароля для веб-интерфейса:
bash
# Редактируем файл с паролем
nano data/web-password.txt

# Перезапускаем приложение
pm2 restart vps-finder
📋 Команды для управления
Команда	Описание
./install.sh	Установка
./update.sh	Обновление
./manage-passwords.sh	Просмотр паролей
npm start	Запуск
npm run update	Обновление данных
pm2 status	Статус
pm2 logs vps-finder	Логи
pm2 restart vps-finder	Перезапуск
🛡️ Безопасность
Двухуровневая защита (мастер-пароль + веб-пароль)

JWT токены для сессий

Rate limiting: 300 запросов/15 минут

Helmet для безопасных HTTP заголовков

XSS защита через санитайзинг

⚠️ Важно
Сохраните пароли в надежном месте при установке

При утере мастер-пароля восстановить его невозможно

Регулярно создавайте резервные копии

Не публикуйте файл .env и data/ в публичный доступ
